## 🎬 Animations Used (All GPU-Optimized)

| Animation | Duration | Usage | Purpose |
|-----------|----------|-------|---------|
| `slideUp` | 0.8s | Hero, Subscribe | Entrance effect |
| `fadeIn` | 1s | Text elements | Staggered reveal |
| `gradientShift` | 15s | Hero, CTA | Continuous engagement |
| `transform` | 0.35s | Hover effects | Interactive feedback |
| `scaleX` | 0.4s | Feature borders | Smooth transitions |

**Performance Impact:** All animations use `will-change: transform` and `backface-visibility: hidden` for 60fps performance.
