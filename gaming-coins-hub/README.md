## 🚀 Quick Start

### Install & Run Locally
```bash
npm install
npm run start
```

Site runs at `http://localhost:3000`

### Deploy to Vercel (Free)
```bash
# 1. Push to GitHub
git init
git add .
git commit -m "Initial commit"
git push origin main

# 2. Go to vercel.com and import the repo
# 3. Done! Auto-deploys on every push
```

## 📊 Analytics to Track

Watch these metrics weekly:
- **Traffic source:** Where visitors come from (Google, Reddit, etc)
- **Bounce rate:** How many leave without scrolling
- **Avg time on page:** Engagement metric
- **CTR in Google:** Click-through rate (topic by topic)
- **Top pages:** Which guides get most traffic
- **AdSense RPM:** Revenue per 1000 impressions

## ⚙️ Customization

### Change Site Name


### Add Your Email
Edit `.env.local`:
```
SUBSTACK_API_KEY=your_key
```

### Update Game Links
Edit `src/pages/index.tsx` (games array) — add your affiliate links

Not using SSH:

If you are using GitHub pages for hosting, this command is a convenient way to build the website and push to the `gh-pages` branch.
