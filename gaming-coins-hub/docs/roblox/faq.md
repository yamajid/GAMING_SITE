# ❓ Roblox Currency FAQ

## Frequently Asked Questions About Robux

### 🎮 Basic Questions

<details>
<summary><strong>What exactly are Robux?</strong></summary>

Robux is Roblox's premium virtual currency used to purchase cosmetics, game passes, developer items, and exclusive features. You can earn small amounts through gameplay or purchase Robux directly.

**Key facts:**
- 1 Robux = 1 unit of currency
- Can be purchased with real money
- Can be earned very slowly through premium membership stipend
- Used for cosmetics ONLY (not pay-to-win)

</details>

<details>
<summary><strong>Do Robux generators, hacks, or unlimited sites work?</strong></summary>

**NO. 100% NOT.**

All "free Robux" generator websites are scams. Here's why:

1. **Technically impossible**: Roblox servers don't allow external currency injection
2. **They steal accounts**: Require login = credentials harvested
3. **They install malware**: Download "generator software" = trojan/spyware
4. **Result**: Your account gets hacked, Robux stolen, account permanently banned

**The scam loop:**
1. User sees "Free Robux!" ad
2. Clicks → visits sketchy website
3. Enters username/password = fields logged
4. "Processing... 5 minutes"
5. Website redirects, nothing happens
6. User's account hacked within 24 hours
7. Roblox bans account for suspicious activity

**If you used a generator:**
- Change password IMMEDIATELY
- Enable 2FA authentication
- Contact Roblox support with evidence
- Run malware scan on your computer

</details>

### 📊 Developer & Income Questions

<details>
<summary><strong>Can you actually make money from Roblox games?</strong></summary>

**YES** but with conditions:

**DevEx (Developer Exchange):**
- Convert Robux you earned → real USD
- Minimum: 100,000 Robux = ~$350 USD
- Requires: 13+ years old, phone verification, 30-day account age
- Rate: ~$0.0035 per Robux (varies)

**Realistic earnings by tier:**

| Creator Level | Entry Revenue | Potential |
|---|---|---|
| New game | $0 first month | $50-500/mo |
| Moderate game | $100-1000/mo | $1000-10K/mo |
| Popular game | $10K+/mo | $100K+/mo |
| Top games | $50K+/mo | $1M+/year |

**Most profitable game types:**
1. Simulators (idle games = easiest monetization)
2. Tycoon games (progression + cosmetics)
3. Roleplay games (cosmetics demand)
4. Competitive games (battle pass style)

</details>

<details>
<summary><strong>What's Roblox's commission on game sales?</strong></summary>

**Roblox takes 30% of all in-game purchases:**
- Players spend 100 Robux
- Roblox takes 30 Robux
- You get 70 Robux

**Then for DevEx:**
- You need to accumulate Robux earned from your games
- Only earned Robux count (not purchased)
- Minimum 100,000 earned Robux to exchange
- Exchange rate: ~$0.0035 per Robux ($350 minimum)

**Example:**
- Your game has 100K players spend $1000 total
- Players spent 100,000 Robux
- Roblox takes 30% = 30,000 Robux
- You get 70% = 70,000 Robux
- But it's only 70K of 100K you need
- Takes time to accumulate enough for payout

</details>

### 🛡️ Safety & Scam Questions

<details>
<summary><strong>What's the #1 Robux scam I should avoid?</strong></summary>

**"Survey sites offering free Robux"** - #1 scam by volume:

**How it works:**
1. See ad: "Complete 3 surveys = 5000 FREE ROBUX!"
2. Click → Survey website loads
3. Complete surveys (takes 30+ minutes)
4. Surveys say "Robux processing..."
5. Never receive Robux
6. Your personal data sold to 3rd parties
7. Start getting spam emails/calls

**What really happened:**
- Your name, email, phone, age, preferences harvested
- Sold to data brokers ($2-10 per record)
- You're on spam lists forever
- Roblox sees fake activity,account banned
- Identity theft risk

**NEVER visit "free currency" sites. Period.**

</details>

<details>
<summary><strong>How do I recognize a Robux scam?</strong></summary>

**Red flags:**

| Red Flag | What it means |
|----------|---------------|
| Requires login | Phishing attempt |
| "Download tool" | Malware delivery |
| Needs credit card | Fraud scheme |
| Email verification | Data harvesting |
| "Limited time!" | Pressure tactic |
| "Too good to be true" offer | It is |
| Unknown URL | Not official |
| Asks for password rebuild link | Account hijack |

**Safe indicators:**
✅ Official Roblox.com domain
✅ In-game redemption only
✅ No login required
✅ No "software download" needed
✅ Posted on official channels (YouTube, Twitter @Roblox)

</details>

<details>
<summary><strong>I accidentally entered my password on a sketchy site. What do I do?</strong></summary>

**IMMEDIATE actions (within 1 hour):**

1. **Change password** (BEFORE they access your account)
   - Go to account settings
   - Choose complex password (uppercase + lowercase + numbers + symbols)
   - Don't reuse old passwords

2. **Enable 2FA**
   - Settings → Security → 2-Step Verification
   - Use authenticator app (not SMS if possible)
   - Save backup codes in safe location

3. **Run malware scan**
   - Download Malwarebytes (free tier okay)
   - Full system scan
   - Restart computer

4. **Monitor your account**
   - Check for unfamiliar purchases
   - Review login history
   - Report suspicious activity immediately

5. **Contact Roblox support** if you see hacking signs

**Prevention going forward:**
- Never share password with anyone
- Always check URL before entering credentials
- Only on official Roblox.com domain
- Verify HTTPS lock icon

</details>

**Can you get Game Passes for free?**

- Events might include free Game Pass days

</details>
