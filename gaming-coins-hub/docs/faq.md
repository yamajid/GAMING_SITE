### 🛡️ Scam Prevention (Critical!)

**The #1 Scam Across ALL Games:**
- Fake "free currency generator" websites
- Ask for login credentials
- Harvest passwords, steal accounts
- Result: Account hijacked, currency stolen, account banned

**How to stay safe:**
1. **NEVER enter password on non-official websites**
2. Use only official game sites or app stores to purchase
3. Be suspicious of "too good to be true" offers
4. Report scam websites to official game support
5. If hacked: Change password IMMEDIATELY, enable 2FA

**[See detailed scam guide](/docs/roblox/faq.md#-safety--scam-questions)** - Learn every scam type (Roblox section)
