1. Website claims "Free Diamonds!"
2. Enter Mobile Legends UID (account ID)
3. Enter password "for verification"
4. "Processing for 5 minutes..."
5. Redirects or asks for credit card
6. Account hacked within hours
7. Hacker changes password, steals Diamonds

**Result for victim:**
- Account hijacked
- Cosmetics/heroes gone
- Permanently banned by Moonton
- Personal data possibly sold

**If you used a generator:**
1. Change password IMMEDIATELY
2. Enable 2FA if available
3. Contact Moonton support with account details
4. Run antivirus scan
5. Monitor banking for fraud

**Safe rule:** Never enter password on non-official sites

</details>

### 🛡️ Scam Questions

<details>
<summary><strong>What's the #1 Diamond scam?</strong></summary>

**"Free Diamond Generator" websites - 90% of scams**

**How victims get caught:**
1. Search: "free mobile legends diamonds"
2. Clicks sponsored ad
3. Website looks official (often fake Moonton logo)
4. Huge banner: "GET 10,000 DIAMONDS FREE!"
5. "Enter your UID to start"
6. User enters UID (account ID) + password
7. Redirects to offer wall or blank page
8. Hours later: Account hacked, Diamonds stolen

**What happens next:**
- Email notifications: "Resources purchased"
- Hacker buys cosmetics and gifts to friends
- Hacker sells account on dark web
- Original owner: Permanently banned (associated with fraud)

**Prevention:**
- NEVER enter password on non-official sites
- Official URL: mobilelegends.moonton.com only
- Real events don't require login
- Moonton never asks for passwords via email

</details>
