## Common Redemption Problems

**"Code has expired"** — The code is no longer valid. Livestream codes expire within 24 hours. Check the date on any code you find online.

**"Code is invalid"** — Double-check for typos. Codes are case-sensitive and spaces matter.

**"Reached redemption limit"** — Each code can only be used once per account.

**"Server not found"** — Make sure you selected the correct server (Asia, America, Europe, TW/HK/MO) from the dropdown before entering the code.
