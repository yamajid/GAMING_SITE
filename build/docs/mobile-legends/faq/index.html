Is Mobile Legends free to play?

YES - 100% free-to-play. NO pay-to-win.

What this means:

  • ✅ All heroes unlockable through gameplay
  • ✅ Cosmetics (skins) = cosmetic only
  • ✅ Equipment purchasable with earned Coins
  • ✅ Competitive matches: Fair skill-based
  • ✅ No "premium only" tools

What you CAN buy with $:

  • Cosmetics (skins, recall animations)
  • Battle pass progression speed
  • Cosmetic emotes
  • Premium cosmetics only

What you CANNOT buy with $:

  • Stronger heroes
  • Better equipment
  • Gameplay advantages
  • Faster hero leveling (minimal)

Bottom line: Spend $0 and compete equally with whales who spend $1000+

Are Diamond generators, hacks, or sites safe?

NO. 100% scams.

How generator scams work:

  1. Website claims "Free Diamonds!"
  2. Enter Mobile Legends UID (account ID)
  3. Enter password "for verification"
  4. "Processing for 5 minutes..."
  5. Redirects or asks for credit card
  6. Account hacked within hours
  7. Hacker changes password, steals Diamonds

Result for victim:

  • Account hijacked
  • Cosmetics/heroes gone
  • Permanently banned by Moonton
  • Personal data possibly sold

If you used a generator:

  1. Change password IMMEDIATELY
  2. Enable 2FA if available
  3. Contact Moonton support with account details
  4. Run antivirus scan
  5. Monitor banking for fraud

Safe rule: Never enter password on non-official sites

📱 Monthly Pass & Events​

What is the Monthly Pass and is it worth the $3.99?

Monthly Pass is a battle pass system costing 128 Diamonds (~$3.99):

What you get:

  • 30 daily missions (cosmetics + currency)
  • 50+ cosmetic items
  • 32-50 Diamonds back (breakeven)
  • Hero fragments
  • Magic dust (upgrade material)
  • Premium recall animations (3-day trials)
  • Emote chests

Is it worth it?

YES if you:

  • Play daily (5+ hours)
  • Complete ALL missions
  • Want cosmetics

NO if you:

  • Play casually
  • Skip days frequently
  • Only want Diamonds (get minimal)

ROI calculation:

  • Cost: $3.99
  • Cosmetics value: $10-20 if purchased separately
  • Diamonds returned: 32-50 (partial refund)
  • Net cost: $1-3 for $10-20 cosmetics
  • Value: 3-10x return

Best strategy: Buy during free Diamond events = maximize value

🛡️ Scam Questions​

What's the #1 Diamond scam?

"Free Diamond Generator" websites - 90% of scams

How victims get caught:

  1. Search: "free mobile legends diamonds"
  2. Clicks sponsored ad
  3. Website looks official (often fake Moonton logo)
  4. Huge banner: "GET 10,000 DIAMONDS FREE!"
  5. "Enter your UID to start"
  6. User enters UID (account ID) + password
  7. Redirects to offer wall or blank page
  8. Hours later: Account hacked, Diamonds stolen

What happens next:

  • Email notifications: "Resources purchased"
  • Hacker buys cosmetics and gifts to friends
  • Hacker sells account on dark web
  • Original owner: Permanently banned (associated with fraud)

Prevention:

  • NEVER enter password on non-official sites
  • Official URL: mobilelegends.moonton.com only
  • Real events don't require login
  • Moonton never asks for passwords via email
Is it safe to buy Diamonds from non-official sellers?

NO. Only buy from official in-game shop:

❌ AVOID:

  • Facebook Marketplace
  • Discord sellers
  • Telegram bots
  • Reddit traders
  • Resale websites
  • "Cheap Diamonds" Discord

✅ SAFE only:

  • In-game shop (direct purchase)
  • Official app store (Apple App Store, Google Play)
  • Official payment methods (credit card, PayPal)

Why third-party is risky:

  • Chargebacks = account ban
  • Stolen account resale = fraud allegations
  • No support from Moonton
  • Personal data theft risk

If account compromised:

  1. Change password
  2. Enable 2FA if available
  3. Contact support: cs@moonton.com
  4. Provide account details and evidence
  5. Hope for recovery (90% denied if purchased fraudulently)
