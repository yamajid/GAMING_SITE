# Frequently Asked Questions

## General Questions

### Are free currency generators real?
No. Every single "free Robux generator," "V-Bucks hack," or "Diamond generator" is a scam. They either steal your account credentials, install malware, or run surveys to make money off you without delivering anything. Roblox, Epic Games, and Moonton have all officially stated that no third-party tool can generate in-game currency.

### Can I really earn free in-game currency without spending money?
Yes — but it takes time and effort. Each game has legitimate methods:
- **Roblox**: Promo codes, game creation, clothing design, affiliate program
- **Fortnite**: Save the World daily quests, Battle Pass self-funding loop
- **Mobile Legends**: Daily login, event rewards, diamonds from events
- **Clash of Clans**: Achievements, Gem Mine, Google Opinion Rewards
- **Genshin Impact**: Daily commissions, Spiral Abyss, version events
- **PUBG Mobile**: Google Opinion Rewards, Microsoft Rewards, events
- **Minecraft**: Microsoft Rewards points converted to Minecoins
- **EA FC 25**: Squad Battles rewards, Transfer Market trading

### How long does it take to see results?
- **Promo codes**: Instant (5–10 minutes)
- **Daily login rewards**: 1–7 days to accumulate meaningful amounts
- **Creating content (games, clothing)**: 1–3 months to first earnings
- **Grinding events**: Varies per event cycle (usually 1–4 weeks)

## Safety

### How do I know if a site is a scam?
Red flags to watch for:
- Asks for your game account password
- Promises "instant" currency with no effort
- Requires you to complete surveys to "unlock" rewards
- Not on an official game domain (roblox.com, epicgames.com, etc.)
- Has terrible grammar and generic stock images

### What should I do if I already used a generator site?
1. Change your game account password immediately
2. Enable two-factor authentication (2FA)
3. Check your account for any unauthorized purchases or items removed
4. Report the site to the game developer
