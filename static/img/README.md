# Images

- `og-image.svg` — Open Graph image used for social sharing previews (1200x630)
- `logo.png` — Site logo (light mode)
- `logo-dark.png` — Site logo (dark mode)
- `favicon.ico` — Browser favicon

## To replace og-image with a custom design:
Upload a 1200x630px JPG or PNG as `og-image.jpg` and update `docusaurus.config.ts`
to reference `/img/og-image.jpg` instead of `/img/og-image.svg`.
